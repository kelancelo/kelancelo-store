# Kelancelo store
Kelancelo store is an e-commerce site inspired by my favorite guitarist Synyster Gates. It uses paypal to process payments.

## Tech stack
These are the technologies used to build this project.
* Auth0
* Bootstrap
* Express
* Heroku
* Node
* Prisma
* PostgreSQL
* React
* Sass
